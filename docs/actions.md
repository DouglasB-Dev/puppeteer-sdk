# Catalogo de acciones

El manual principal para construir archivos `projects/<project>/steps/*.js` esta en `docs/guide.md`.

Esta pagina queda como indice rapido de acciones disponibles:

- Navegacion: `goto`, `goBack`, `goForward`, `reload`
- Espera: `wait`, `waitForSelector`, `waitForNavigation`, `waitForFunction`, `waitForNetworkIdle`
- Interaccion: `click`, `type`, `select`, `hover`, `scroll`, `press`, `uploadFile`
- Extraccion: `extractText`, `extractAttribute`, `extractHTML`, `extractList`, `extractTable`, `screenshot`
- Validacion: `validateUrl`, `validateSelector`, `validateText`, `validateCondition`, `validateData`
- Contexto: `setVariable`, `setData`
- Logs: `consoleLog`, `log`
- Control de flujo: `conditional`, `loop`, `runSteps`

Consulta `docs/guide.md` para ver parametros, ejemplos y patrones de uso.
